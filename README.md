# Sistema de Animais (Projeto de Estudo)

Este é um projeto simples desenvolvido enquanto aprendo os conceitos básicos de **Java** e **Programação Orientada a Objetos (POO)**.

O objetivo do sistema é praticar:

* uso de **classes e subclasses**
* **herança**
* **classes abstratas**
* **sobrescrita de métodos**
* interação simples com o **usuário via terminal**

No programa, o usuário pode escolher um animal (Cachorro ou Gato), informar seu nome e ver o animal emitir um som e executar a ação de dormir.

Por ser um projeto de estudo, o foco está na prática dos conceitos, e não na complexidade.
Futuramente pretendo evoluir para projetos maiores e mais completos.
